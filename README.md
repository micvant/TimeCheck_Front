# TimeCheck Frontend

Веб‑клиент учета времени (React + Vite + IndexedDB).

## Требования
- Node.js 18+

## Запуск
```bash
cd /home/micvant/PycharmProjects/TimeCheck_frontend
npm install
npm run dev
```

Открой `http://localhost:5173`.

## Переменные окружения
API адрес можно переопределить:
```bash
export VITE_API_BASE="http://localhost:8000"
```

## Авторизация
Перед синхронизацией зарегистрируйся или войди в правом верхнем углу.
